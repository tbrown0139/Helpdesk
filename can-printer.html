Important: Starting in 2025, all computers that want to access and print to this printer will require the driver to be installed and authenticated via a Knovon account or an Auth ID.

Follow these step-by-step instructions for your Canon Color imageCLASS MF741Cdw printer. Select the topic you need help with below:

Install Drivers (Windows)
Install Drivers (macOS)
Scan Documents
Make Copies

For Windows:

  1. Visit the Canon Driver Support page.
  2. Select your Windows version from the dropdown menu.
  3. Download the "MF743Cdw/MF741Cdw MFDrivers" package.
  4. Locate the downloaded file and double-click it to start installation.
  5. Follow the on-screen prompts to complete the setup.

For macOS:

  1. Visit the Canon Driver Support page.
  2. Select your macOS version from the dropdown menu.
  3. Download the "Mac MF Printer Driver" package.
  4. Locate the downloaded file (usually in the Downloads folder).
  5. Double-click the installer and follow the on-screen instructions.
  6. Ensure the printer is added in "System Preferences > Printers & Scanners."

Scan Documents and Email Them:

  1. Place the document you want to scan on the printer's scanner bed or in the automatic feeder.
  2. On the printer's touchscreen, select Scan.
  3. Choose Email as the destination.
  4. Enter your email address in the prompt or select it from the address book.
  5. Press Start to begin scanning. The document will be sent to your email.

Make Copies:

  1. Place the document you want to copy on the scanner bed or in the automatic feeder.
  2. On the printer's touchscreen, select Copy.
  3. Set the number of copies and adjust settings such as color or black and white.
  4. Press Start to begin copying.
