Canon Printer Driver Installation

Knovon Helpdesk

Important: Starting in 2025, all computers that want to access and print to this printer will require the driver to be installed and authenticated via a Knovon account or an Auth ID.

Follow these step-by-step instructions to install your Canon Color imageCLASS MF741Cdw printer drivers. Select your operating system below:

Windows
macOS

For Windows:

  1. Visit the Canon Driver Support page.
  2. Select your Windows version from the dropdown menu.
  3. Download the "MF743Cdw/MF741Cdw MFDrivers" package.
  4. Locate the downloaded file and double-click it to start installation.
  5. Follow the on-screen prompts to complete the setup.

For macOS:

  1. Visit the Canon Driver Support page.
  2. Select your macOS version from the dropdown menu.
  3. Download the "Mac MF Printer Driver" package.
  4. Locate the downloaded file (usually in the Downloads folder).
  5. Double-click the installer and follow the on-screen instructions.
  6. Ensure the printer is added in "System Preferences > Printers & Scanners."

Authentication Requirements:

To use this device, you need to authenticate with an ID and PIN. The default settings are:

  • ID: The last 7 digits of your phone number.
  • PIN: The last 4 digits of your phone number.

You can update these settings in the Knovon One User Portal.
